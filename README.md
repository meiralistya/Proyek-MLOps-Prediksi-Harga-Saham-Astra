# Proyek-MLOps-Prediksi-Harga-Saham-Astra
Proyek ini merupakan implementasi end-to-end Machine Learning Operations untuk memprediksi harga saham PT Astra International Tbk.  Fokus utama proyek ini bukan pada kompleksitas model, melainkan pada workflow MLOps, mulai dari data ingestion, preprocessing, training, deployment, hingga monitoring sistem machine learning.
